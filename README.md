# Rapport_Suivi_Ventes
Création d'un rapport de suivi des ventes d'une entreprise qui fait l'achat de matériels informatiques

# Objectifs:
Le but est de créer un rapport sur les ventes avec 2 principaux onglets
1.Le 1er onglet pour le suivi global des ventes
Ici, on va analyser le nombres de commandes, la quantité vendue, le chiffre d'affaire

2.Le 2ème onglet pour le suivi des commandes annulées
Ici on va crér: 
-Des mesures DAX pour indiquer le pourcentage des commandes qui ont été annulées, 
-Les courbes d'évolution, 
-Des trimaps pour indiquer les typologies des produits ayant le plus de commandes annulées

# Étapes Pour atteindre ces ojectifs
Pour atteindre ces objectifs, nous avons fait les étapes suivantes:
1.Data cleaning avec Power Query
Ici nous avons nettoyer, filtrer et créer des colonnes calculées

2.Modélisation en étoile
Ici nous avons transformer une table brute en table de faits et de dimension Power Model claires et performant

3.Créer des mesures DAX
Ici nous avons créer des calculs dynamiques qui réagissent aux filtres appliqués dans le rapport

4.Personnaliser le design

5.Ajouter des segments pour filtrer sur les données du tableau et des ensembles de graphique

6.Ajouter des menus pour pouvoir naviguer facilement d'un item à un autre

7.Ajouter des infobulles personnalisés
Ici nous avons rajouter au survol d'une personnalisation une autre personnalisation
